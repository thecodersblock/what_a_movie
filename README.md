# README

## API Endpoints

### Postman Doc: https://documenter.getpostman.com/view/4790043/SVYwJFnZ?version=latest

- ##### Login User
URL: 
`POST /user`
Body:
```json
{
  username: "Raj"
}
```
Response (if user exists):
```
status: 200
{
  code: 'Success'
}
```
Response (if user doesn't exist):
```
status: 404
{
  code: "Not Found"
}
```

- ##### Register User
URL: 
`PUT /user`
Body:
```json
{
  username: "Raj"
}
```
Response (if user exists):
```
status: 400
{
  code: 'Already exists'
}
```
Response (if user doesn't exist):
```
status: 200
{
  code: "Success",
  username: "Raj"
}
```
